# RadFlow Studio — Product Overview

RadFlow is a **visual low-code IDE** built with Flutter (desktop/Windows). It lets users compose Flutter UIs by dragging and dropping widgets onto a canvas, editing their properties in a side panel, and exporting the result as ready-to-run Dart/Flutter code.

## Core Capabilities

- Multi-page project management (add, rename, remove pages)
- Widget tree with parent/child relationships (drag-to-reparent)
- Live canvas preview with device presets (phone, tablet, desktop)
- Properties panel for editing widget props (colors, text, layout, etc.)
- Dart code generation from the visual widget tree
- JSON-based project serialization (import/export)

## Target User

Developers and designers who want to prototype or build Flutter apps visually without writing boilerplate layout code by hand.

## Language / Locale

UI strings and code comments are written in **Brazilian Portuguese (pt-BR)**. New code should follow the same convention.
